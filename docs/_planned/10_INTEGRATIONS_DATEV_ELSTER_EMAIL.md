# Integrations: DATEV, ELSTER, Email

## Status: Not Implemented Yet

This document tracks planned integrations for:

- **DATEV** (German accounting export)
- **ELSTER** (German tax authority submission)
- **Email** (transactional and notification emails)

### Planned Scope

- DATEV: Export accounting data in DATEV-compliant format for tax advisors and auditors.
- ELSTER: Submit VAT and tax filings electronically via ELSTER API (planned for Phase 4).
- Email: Integrate with SMTP or transactional email provider for notifications, alerts, and document delivery.

> Implementation is out of current project scope. This document will be updated when integration work begins.
